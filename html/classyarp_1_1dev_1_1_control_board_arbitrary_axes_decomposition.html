Detailed Description

Class storing the decomposition of a subset ot the total remapped axes of the remapped controlboard in the corresponding subsets of the axes of the SubControlBoard.

This class is meant to be used when implementing multiple joint methods (the one with the signature const int n_joints, const int *joints, double *dpos )
